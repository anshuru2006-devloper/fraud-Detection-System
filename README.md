# Fraud Detection System | Group Project
PHP + MySQL web app to detect phishing URLs using Google Safe Browsing API.

## Development Team
This project was developed as part of BCA curriculum. The complete coding and implementation was done by:

- Anjali Kumari [@anshuru2006-devloper](https://github.com/anshuru2006-devloper) - Frontend Development, Backend Development, Database Design, API Integration, Project Lead

*Note: Project documentation and report was handled by other team members as per college requirements.*

## My Key Contributions
- Integrated Google Safe Browsing API to enable real-time URL scanning and classify malicious links
- Designed normalized MySQL database schema with proper indexing for efficient data retrieval
- Developed secure user authentication system using PHP Sessions for login/logout functionality
- Implemented AJAX for asynchronous URL scanning to improve user experience without page reloads

## Tech Stack
**Backend:** PHP, MySQL | **Frontend:** HTML5, CSS3, JavaScript, AJAX, Bootstrap
**API:** Google Safe Browsing | **Tools:** XAMPP, Git
